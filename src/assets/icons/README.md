# Icons

`Icon.tsx` holds temporary geometric placeholder icons (24x24, currentColor strokes).

The product owner will deliver custom icons later — see `ICONS_REQUIRED.md` at the repo root for the full required list. When the real set arrives, replace the shapes in `Icon.tsx` (keep the `IconName` keys stable so no call sites change).

Rules: SVG only, `currentColor` strokes, no emojis anywhere.
